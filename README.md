# Alzheihmer-Detection-and-Progress-Prediction

How to Open the Project :-
1. Clone the repository using command (git clone https://github.com/Sahil-Nitjit/Alzheihmer-Detection-and-Progress-Prediction.git)
2. Open the Project (cd Alzheihmer-Detection-and-Progress-Prediction)
3. Install Libraries (pip install streamlit tensorflow numpy pillow opencv-python plotly)
4. Start the Project (streamlit run app.py)
